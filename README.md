# JavaScript Todo List Application

A simple, responsive todo list application built with vanilla JavaScript that allows users to add, edit, delete, and filter tasks.

## Features

- **Add Tasks**: Type a task in the input field and click "Add Task" or press Enter
- **Edit Tasks**: Click the edit (pencil) icon to modify any task
- **Delete Tasks**: Click the delete (trash) icon to remove tasks
- **Mark as Complete**: Check the checkbox to mark tasks as completed
- **Filter Tasks**: View all tasks, only pending tasks, or only completed tasks
- **Task Statistics**: See the total, completed, and pending task counts
- **Persistent Storage**: Tasks are saved in the browser's localStorage
- **Responsive Design**: Works well on both desktop and mobile devices
